**Ucount**
===

# 宅记

***
做这个APP纯粹是因为不希望被冗余的功能和广告烦恼，只要能够满足自己的日常所需就好。

整体界面参考的是Timi记账。因为是新手写Android，有很多方面都做得不是很好，后续还会进行迭代更新。

<p align="right">2017-03-15</p>
